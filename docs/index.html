File Naming Convention

CPI-BGD-{ProgramCode}-{DocumentType}-{YYYYMM}-v{Version}.{ext} Examples: CPI-BGD-HOP-MonthlyReport-202604-v1.pdf CPI-BGD-HCV-SOP-202601-v2.docx CPI-BGD-CVD-TrainingManual-202603-v1.pptx
