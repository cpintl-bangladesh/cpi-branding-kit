# CPI Icon Usage Guide
## Community Partners International — Icons & Visual Symbols

---

## OFFICIAL CPI ICONS (from brand guide)

CPI has four official icons used in branded materials, available in two styles:
- **Non-solid (outline):** CPI Red stroke circle with white background icon inside
- **Solid (filled):** CPI Red filled circle with white icon inside

### Official Icon 1 — Community / Partnership
- **Symbol:** Circular arrangement of people/hands connected
- **Represents:** Community partnership, network, collaboration, CHW teams
- **Use in:** HOP materials, community engagement content, partnership reports

### Official Icon 2 — Family / Women & Children
- **Symbol:** Family group — adult figure with children
- **Represents:** Maternal & child health, gender equity, family services
- **Use in:** Maternal health, child health, gender-based violence content

### Official Icon 3 — Health / Care
- **Symbol:** Cupped hands holding a plant/seedling or heart
- **Represents:** Health care, nurturing, growth, healing
- **Use in:** Health program materials, health post content, program overviews

### Official Icon 4 — Shelter / Home
- **Symbol:** House with person/care indicator
- **Represents:** Shelter, safety, humanitarian response
- **Use in:** Emergency response, shelter programs, humanitarian materials

## RECOMMENDED SUPPLEMENTARY ICONS

For program materials and reports, use icons from consistent free icon sets.

**Recommended sets (free, consistent style):**
- **Material Design Icons** — https://fonts.google.com/icons
- **Feather Icons** — https://feathericons.com
- **Heroicons** — https://heroicons.com

**Apply CPI styling:**
- Fill with CPI Red `#D91E4D` for primary use
- Fill with CPI Purple `#41273B` for secondary use
- White icons inside CPI Red or Purple filled circles for buttons/badges

## ICON SIZING STANDARDS

| Context | Icon Size |
|---------|----------|
| Body text inline icon | 16–18px |
| List bullet icon | 20–24px |
| Card/feature icon | 40–60px |
| Section header icon | 48–64px |
| Infographic large icon | 80–120px |
| Name card icon | 10–12pt |

## INFOGRAPHIC ICON USAGE

For infographic slides and pages:

### Stat Infographic Structure:
```
[Large Icon — 80px, white on CPI Red circle]
[Large Number — Arial Bold, 48pt, CPI Red]
[Label — Arial Regular, 14pt, CPI Purple]
[Short description — Arial Regular, 11pt, CPI Black]
```

### Process Flow Icons:
- Use same icon set throughout
- Connect with CPI Red arrows (→)
- Number steps: 1, 2, 3... in CPI Red circles with white numbers

### Indicator Tracker Icons:
```
✅ = Target met / exceeded (CPI Blue / Green)
⚠️ = Below target, on track (Orange/Yellow)  
❌ = Critical gap (CPI Red fill)
➡️ = In progress / ongoing
```

---

## ICON DON'TS

- ❌ Never use different icon styles in the same document (mix outline and flat)
- ❌ Never use clipart or pixelated icons
- ❌ Never use emoji as primary icons in professional reports
- ❌ Never use icons that depict stereotypes or culturally insensitive symbols
- ❌ Never resize icons disproportionately
- ❌ Never use more than one icon per bullet point in a list
- ❌ Never use icons as logo replacements

## COLOR APPLICATION FOR ICONS

| Context | Icon Style |
|---------|-----------|
| On white background | CPI Red `#D91E4D` solid fill |
| On CPI Red background | White `#FFFFFF` |
| On CPI Purple background | White `#FFFFFF` |
| Secondary/muted context | CPI Mid Grey `#948794` |
| Highlighted / active | CPI Red `#D91E4D` filled circle + white icon inside |
| Disabled / inactive | CPI Light Grey `#D0C4C5` |
