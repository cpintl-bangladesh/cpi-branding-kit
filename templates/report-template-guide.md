```
Report Date: [DD Month YYYY]
Prepared by: [Name, Title]
Reviewed by: [Supervisor Name, Title]

[Horizontal rule: CPI Red, 1pt]

Community Partners International Bangladesh
Rabbani Building (4th Floor), Opposite S.K. Tower
Kurushkul Bypass Road, Cox's Bazar, Bangladesh
Tel: +880 15 3420 6479 | healthbgd@cpintl.org | www.cpintl.org
```

### Section 1: Executive Summary (max 200 words)
Key points only:
- Top 3 achievements
- Top challenge
- Key action for next month

### Section 2: Program Context
- Target population and geographic coverage
- Operational period
- Brief program objective

### Section 3: Key Achievements (the main body)
- One paragraph per major achievement
- Link to indicators where possible
- Include anecdotal/case evidence

### Section 4: Indicator Performance Table
```
TABLE: Indicator Performance — [Month Year]

| # | Indicator | Target (Cumulative) | Target (Monthly) | Achieved (Month) | Achieved (Cumulative) | % of Monthly Target | Remarks |
|---|-----------|--------------------|-----------------|-----------------|-----------------------|--------------------|----|
| 1 | | | | | | | |
```
**Formatting:**
- Header: CPI Purple background, white Arial Bold 10pt
- Cells meeting/exceeding target: Light green fill
- Cells below target: Light yellow fill
- Cells critically below (<50%): Light red fill

### Section 5: Challenges & Mitigation
| Challenge | Impact | Mitigation Action | Responsible | Timeline |
|-----------|--------|------------------|-------------|---------|

### Section 6: Field Story / Case Study
- Title: [Name]'s Story (use real name with consent OR anonymize)
- 150–300 words
- Photo if available (captioned and consented)
- Connects individual story to program impact

### Section 7: Next Month Plan
- Planned activities
- Expected outputs
- Any deviations from workplan to flag

### Annexes (as needed):
- A: Detailed data tables
- B: CHW activity logs summary
- C: Meeting minutes
- D: Photos (captioned)

## ANNUAL REPORT STRUCTURE

**Format:** Professional designed PDF (InDesign preferred OR Word)  
**Length:** 20–40 pages

### Sections:
1. Cover (full-bleed photo + CPI branding)
2. Table of Contents
3. Letter from Program/Country Director
4. Highlights at a Glance (infographic page with key stats)
5. Context: The Challenge We Face
6. Our Response (by thematic area):
   - Health
   - WASH / Humanitarian
   - Community Outreach
7. Impact Stories (3–5 stories with photos)
8. Data & Indicators (charts and tables)
9. Partners & Coordination
10. Financials / Accountability
11. Looking Ahead
12. Acknowledgements & Donor List (if appropriate)
13. Back Cover (contacts, website, social)

**Design specs for Annual Report:**
- All Raleway font (printed design)
- Full-bleed photo spreads for story sections
- Infographic pages using CPI color palette charts
- Pull quotes in large text with CPI Red accent
- Two-column layout for body content
- Data visualization in CPI color palette

## DATA VISUALIZATION STANDARDS FOR REPORTS

### Chart Types by Data:
| Data Type | Recommended Chart | CPI Color(s) |
|-----------|------------------|-------------|
| Progress over time | Line chart | CPI Red primary line |
| Category comparison | Horizontal bar | CPI Red + Purple bars |
| Part of whole | Pie / Donut | CPI color set |
| Two-variable comparison | Grouped bar | CPI Red + Blue |
| Distribution | Histogram | CPI Purple fill |
| Geography | Choropleth map | CPI Red to Light Grey gradient |
| Cascade (e.g., 90-90-90) | Waterfall/Step chart | CPI Red, Purple, Blue |

### Chart Formatting Rules:
- **Title:** Arial Bold, 13pt, CPI Purple — above chart
- **Axis labels:** Arial Regular, 10pt, CPI Black
- **Data labels:** Arial Bold, 10pt — on bars/points
- **Legend:** Arial Regular, 10pt — below chart
- **Grid lines:** 0.5pt, CPI Light Grey — horizontal only
- **Source note:** Arial Regular, 8pt, CPI Mid Grey — bottom-right
- **No 3D charts** — always use flat/2D charts
- **No decorative gradients** — solid fills only (exception: CPI Red gradient in logo mark)

### HCV Cascade Chart Example:
```
Cascade Steps:
  Step 1: Estimated HCV+ population → CPI Red #D91E4D
  Step 2: Tested (90% target) → CPI Purple #41273B
  Step 3: Diagnosed/confirmed → CPI Blue #4298B5
  Step 4: Treatment initiated (90%) → CPI Purple Secondary #615E9B
  Step 5: SVR achieved (90%) → CPI Mid Grey #948794

Format: Horizontal bar or stacked step-down waterfall
```

## REPORT QUALITY CHECKLIST

### Before Submitting Any Report:
- [ ] Cover page complete (title, period, preparer, date, CPI contacts)
- [ ] CPI logo on cover and header
- [ ] Correct font used (Arial for Word; Raleway for designed PDF)
- [ ] All indicator targets referenced from approved M&E plan
- [ ] Data cross-checked with raw data source (DHIS2, KoboToolbox, registers)
- [ ] All photos captioned and consent documented
- [ ] Spell check and grammar review completed
- [ ] Tables formatted with CPI color header rows
- [ ] File name follows naming convention
- [ ] Reviewed and signed off by supervisor before sending
- [ ] PDF version created for distribution
- [ ] Filed in program Google Drive under correct folder
