# CPI Document Template Guide
## Word / PDF / Google Docs — Compliant Document Creation

---

## WHEN TO USE WHICH DOCUMENT TYPE

| Type | Use For | Font |
|------|---------|------|
| **Microsoft Word (.docx)** | Letters, SOPs, reports, meeting minutes, field guides | Arial |
| **Google Docs** | Collaborative drafts, shared working documents | Arial |
| **PDF (exported)** | Final distribution versions of any document | Arial (from Word) or Raleway (from design software) |
| **Adobe InDesign / Illustrator** | Professional printed reports, brochures, posters | Raleway |
| **PowerPoint** | Presentations, donor briefings, training | Arial |

---

## WORD / GOOGLE DOC SETUP

### Page Setup:
- **Size:** A4 (210 × 297mm) for Bangladesh/international; Letter (8.5 × 11in) for US
- **Margins:** 2.5cm (1 inch) all sides — or 2cm top/bottom, 2.5cm left/right for two-sided print
- **Orientation:** Portrait (default); Landscape only for wide tables/charts

### Styles Configuration (Arial System Font):

```
Normal / Body Text:
  Font: Arial Regular, 11pt
  Color: #2D2926 (CPI Black)
  Line spacing: 1.5
  Space after paragraph: 6pt

Heading 1:
  Font: Arial Bold, 18pt
  Color: #D91E4D (CPI Red)
  Space before: 18pt, Space after: 6pt
  Keep with next: Yes

Heading 2:
  Font: Arial Bold, 14pt
  Color: #41273B (CPI Purple)
  Space before: 12pt, Space after: 4pt

Heading 3:
  Font: Arial Bold, 12pt
  Color: #41273B (CPI Purple)
  Space before: 8pt, Space after: 2pt

Caption / Figure label:
  Font: Arial Regular, 9pt
  Color: #948794 (CPI Mid Grey)
  Italic: Yes

Table Header Row:
  Font: Arial Bold, 10pt
  Color: White #FFFFFF

  Row 1: White background

  Font: Arial Regular, 8pt
```

## STANDARD DOCUMENT STRUCTURES

### 1. SOP (Standard Operating Procedure)

```
COVER PAGE:
  - CPI Logo (top-left)
  - "STANDARD OPERATING PROCEDURE" (Arial Bold, 14pt, CPI Red)
  - SOP Title (Arial Bold, 20pt, CPI Purple)
  - Program / Department
  - SOP Number: [e.g., CPI-BGD-HOP-SOP-001]
  - Version: [e.g., Version 2.0]
  - Effective Date: [Month Year]
  - Review Date: [Month Year]
  - Approved by: [Name, Title]

SECTIONS:
  1. Purpose
  2. Scope
  3. Definitions and Abbreviations
  4. Roles and Responsibilities
  5. Procedure (numbered steps)
  6. Related Documents
  7. Revision History (table)

FOOTER on all pages: CPI logo + SOP number + version + page number
```

### 2. Program/Donor Report

```
COVER:
  - CPI Logo
  - Program Name
  - Reporting Period (e.g., Q1 2026: January–March 2026)
  - Prepared by: Name, Title
  - Date
  - Donor / Audience

SECTIONS:
  1. Executive Summary (max 1 page)
  2. Program Context
  3. Key Achievements (with indicators table)
  4. Challenges and Mitigation
  5. Case Study / Field Story
  6. Financial Summary (if applicable)
  7. Next Quarter Plan
  8. Annexes

TABLE: Indicator Performance Table
  Columns: Indicator | Target | Achieved | % Achieved | Remarks
  Header: CPI Purple background, white Arial Bold
  Data: alternating white / light grey rows
```

### 3. Meeting Minutes

```
HEADER:
  CPI Logo | MEETING MINUTES

  Meeting Title: [e.g., Monthly Health Team Review]
  Date: [DD Month YYYY]
  Time: [HH:MM – HH:MM, timezone]
  Location: [Office / Remote / Platform]
  Facilitated by: [Name]
  Note taker: [Name]

ATTENDEES TABLE:
  Name | Role | Organization | Present?

AGENDA ITEMS:
  (numbered, each with: Discussion / Decision / Action)

ACTION ITEMS TABLE:
  # | Action | Responsible | Deadline | Status

NEXT MEETING: [Date/time]
FOOTER: CPI logo | healthbgd@cpintl.org | www.cpintl.org
```

### 4. Concept Note / Proposal

```
COVER:
  CPI Logo + Partner logo (if co-submission)
  Project Title
  Submitted to: [Donor/Authority]
  Submitted by: Community Partners International Bangladesh
  Date
  Contact: [name, email, phone]

SECTIONS:
  1. Background and Problem Statement
  2. Proposed Intervention
  3. Target Population and Geographic Focus
  4. Implementation Approach
  5. Expected Outcomes and Indicators
  6. Budget Summary
  7. Organizational Capacity
  8. Risks and Mitigation
```

### 5. Training Manual / Field Guide

```
COVER:
  CPI Logo
  "TRAINING MANUAL" or "FIELD GUIDE"
  Title
  Program | CPI Bangladesh
  Audience (e.g., "For Community Health Workers")
  Version + Date

STRUCTURE:
  - Module-based (Module 1, 2, 3...)
  - Each module: Learning Objectives + Content + Key Messages + Quiz/Exercise
  - Use callout boxes for key points, warnings, tips
  - Simple tables for protocols and reference
  - Illustrations/photos with captions
```

## PDF EXPORT SETTINGS

When exporting Word/Google Doc to PDF:
- **Quality:** High (300 DPI equivalent)
- **Embed fonts:** Yes
- **PDF/A compliance:** Recommended for archival
- **Bookmarks:** Map from document headings
- **File name format:** `CPI-BGD-[ProgramCode]-[DocumentType]-[YYYYMM]-v[Version].pdf`

**Naming convention examples:**
- `CPI-BGD-HOP-MonthlyReport-202604-v1.pdf`
- `CPI-BGD-HCV-SOP-202601-v2.pdf`
- `CPI-BGD-CVD-TrainingManual-202603-v1.pdf`
