## How to Use This Kit

### For AI Agents:
1. Upload the entire ZIP or clone this repository
2. Start with `AI_AGENT_QUICKSTART.md` for immediate task routing
3. Load `SKILL.md` as the master reference
4. Reference specific sub-files based on the task (see quickstart guide)

### For Human Users:
1. Start with `SKILL.md` to understand the full scope
2. Navigate to the relevant folder for your task type
3. Follow the template guides for your document type
4. Use `assets/copy-blocks.md` for pre-approved text

## Brand Philosophy

> *"Empowerment and localization are hardwired into Community Partners International. Ninety-seven percent of our workforce is from the countries where we operate."*

CPI's brand reflects its commitment to communities as **empowered actors and equal partners** — not passive recipients. All materials created using this kit should reflect dignity, resilience, and hope.
