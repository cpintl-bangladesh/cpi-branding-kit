# CPI Logo Rules & Usage Guide
## Community Partners International — Official Logo Standards

**Source:** CPI Brand Identity & Standards Designer's Guide (Official)

---

## LOGO VARIATIONS

### 1. Primary Logo (Full Color, Horizontal)
- **When to use:** The default for the vast majority of contexts — documents, presentations, reports, emails, banners, posters
- **Components:** Butterfly/leaf mark (red gradient) + "CPI" wordmark (CPI Purple) + "COMMUNITY PARTNERS INTERNATIONAL" stacked text (CPI Purple)
- **Background:** White or very light backgrounds only

### 2. Primary Logo WITH Integrated Tagline
- **When to use:** Official letterheads, stationery, formal documents
- **Tagline below:** *Empower communities. Transform lives.*
- **Tagline font:** Raleway Medium, CPI Purple or CPI Red
- **Minimum size:** 0.8 cm height

### 3. Secondary Logo (Vertical / Stacked)
- **When to use:** Space-constrained contexts where the horizontal logo does not fit — social media profile thumbnails, small badge areas, narrow columns
- **Layout:** Butterfly mark centered above, "CPI" below, "COMMUNITY PARTNERS INTERNATIONAL" stacked beneath
- **Minimum size:** 0.9 cm height

### 4. White / Reversed Logo
- **When to use:** When placed on any approved CPI brand-colored background (Purple, Red, Blue, Grey, Black)
- **All elements:** Pure white `#FFFFFF`
- **Never use:** On photographic backgrounds or non-CPI brand colors

### 5. Tagline Standalone
- **When to use:** As a supporting element in branded materials where logo is already present; or on branded merchandise
- **Text:** *Empower communities. Transform lives.*
- **Color options:** CPI Purple | CPI Red | White (on CPI background colors)
- **Font:** Raleway Medium

---

## MINIMUM SIZE RULES
> Never use the CPI logo below these dimensions:

| Logo Version | Minimum Height |
|-------------|---------------|
| Primary Logo (horizontal) | **0.5 cm** |
| Secondary Logo (vertical) | **0.9 cm** |
| Primary Logo with Tagline | **0.8 cm** |

For digital use, the equivalent minimum is **20px height** for primary logo.

---

## CLEAR SPACE (EXCLUSION ZONE)
> Maintain clear space equal to the **x-width of the letter "P" in the CPI logo** on all four sides of the logo. No text, graphics, images, or other elements may enter this zone.

**Practical approximation:**
- At 5cm logo width → minimum 0.4cm clear space on each side
- At 10cm logo width → minimum 0.8cm clear space on each side
- Scale proportionally with logo size

---

## LOGO PLACEMENT ON PAGE
The CPI logo may be placed in any of these four positions:
1. **Top Left** ← most common, preferred
2. **Top Right**
3. **Bottom Left**
4. **Bottom Right**

> The logo should **not** be centered unless specified by a specific template (e.g., some covers or banners).

## LOGO MISUSE — STRICTLY PROHIBITED
The following are **NEVER acceptable**:

| ❌ Violation | Description |
|-------------|-------------|
| Proportion change | Stretching or squishing the logo horizontally or vertically |
| Below minimum size | Using logo smaller than specified minimum |
| Busy background | Placing logo on a photographic or patterned background |
| Rotation | Tilting or rotating the logo at any angle |
| Wrong background color | Placing on unapproved colored backgrounds (orange, green, yellow, etc.) |
| Color change | Altering any element of the logo color |
| Distortion | Any skewing or perspective effects |
| Font change | Recreating the wordmark in a different font |
| Drop shadows | Adding shadows, glows, or effects to the logo |
| Outline only | Using an outlined/ghost version not in brand kit |
| Logo in text | Substituting the logo mark mid-sentence in text |

---

## CO-BRANDING RULES
When CPI appears alongside partner/donor logos:
- Use **Primary Logo without tagline** (unless specifically required otherwise)
- Give equal visual weight to all logos in the array
- Maintain CPI logo's clear space even within a logo row
- Preferred separator between logos: thin vertical line in CPI Mid Grey
- CPI logo should be on the LEFT unless otherwise specified by partnership agreement

## LOGO FILE FORMATS & WHEN TO USE

| Format | When to Use |
|--------|------------|
| **SVG** | Web, responsive digital, any scalable screen use |
| **PNG (transparent)** | Digital documents, Word, PowerPoint, email signatures, web |
| **JPG** (on white) | When transparency not supported |
| **PDF** | Print-ready files, high-resolution printing |
| **EPS / AI** | Professional print work, large format, signage |

**Color modes:**
- **RGB** → All digital/screen use
- **CMYK** → All print materials

## LOGO ELEMENT DESCRIPTION (for recreation reference only)
The CPI logo mark is a stylized **butterfly/leaf form** with:
- Three leaf/wing shapes in a gradient from deep red-pink to bright red (`#D91E4D`)
- A curved black stroke separating the leaf elements
- The mark overlaps the "C" of the CPI wordmark slightly

The wordmark "CPI" uses a custom heavy serif weight in **CPI Purple `#41273B`**.
The descriptor "COMMUNITY PARTNERS INTERNATIONAL" is in a lighter, spaced sans-serif in **CPI Purple `#41273B`**, separated from "CPI" by a vertical rule.

> **Do NOT attempt to recreate the logo from description. Always use the official logo files provided in the brand kit.**
